# Agent Memory Protocol

## 写入条件

只有以下信息值得跨会话保存：

- 用户明确或反复表现出的稳定偏好。
- 有理由和影响范围的决策。
- 已验证的最佳实践与失败教训。
- 可复用工作流。
- 项目的真实进展、阻塞、风险和下一步。

## 写入位置

- 偏好、经验、决策、工作流：`97-AI-Memory`。
- 稳定背景和操作规则：`98-AI-Context`。
- 项目状态：`06-Projects/<项目>/Project-Status.md`。

## 禁止

- 保存完整聊天记录。
- 保存一次性寒暄、临时猜测或未经确认的个人信息。
- 重复写入已有事实；优先更新原条目并保留日期。
