# Web Clipper Setup

模板文件：[[Web Clipper/Research Capture.json]]。

## 已配置

- 目标目录：`00-Inbox/Downloaded`。
- 自动保存：标题、网站、原始链接、作者、发布时间、采集时间、完整正文。
- 不启用 Interpreter，不自动总结或改写。

## 浏览器中一次性操作

1. 安装官方 Obsidian Web Clipper 扩展。
2. 打开扩展设置 → Templates → Import。
3. 导入 `98-AI-Context/Web Clipper/Research Capture.json`。
4. 选择本 Vault，并用任意文章测试一次。

导入后，资料将进入 [[00-Inbox/Research Collection Workflow]]。
