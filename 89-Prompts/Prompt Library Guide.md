# Prompt Library Guide

## 原则

- Prompt 是可复用资产，不是聊天记录。
- 一个 Prompt 只解决一个稳定任务。
- 明确角色、上下文、输入、步骤、约束、输出格式和验收标准。
- 修改时更新版本与测试记录；失败案例也要保留。
- 涉及事实更新时要求 Agent 给出日期与来源。

## 生命周期

draft → tested → production → deprecated。

## 分类

按主要用途放入唯一目录；跨领域能力通过 `topic/...` 标签与 Topic Hub 连接。
