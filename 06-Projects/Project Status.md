# Project Status

每个项目放在 `06-Projects/<项目名>/`，并包含独立的 `Project-Status.md`。新建时复制 [[90-Templates/Project Template]]。

## 项目规范

- 当前状态：idea / active / paused / blocked / done。
- 已完成事项：只记录可验证结果。
- 待办事项：使用 Markdown 任务列表。
- 下一步：保持一个明确动作。
- 风险：写触发条件和应对方式。
- 决策：记录日期、决定与理由。
