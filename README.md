## 最小工作流

1. 网页或资料进入 `00-Inbox/Downloaded`。
2. 运行 Research Cleaner，输出到 `00-Inbox/Cleaned`。
3. 人工快速确认主分类与标签，再移入 `04-Research`。
4. 在 Topic Hub 中通过标签和反向链接聚合。
5. 研究结论进入内容、项目、AI Memory 或 AI Context。
